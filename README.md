Node.js implementation of [CashBill](https://www.cashbill.pl/download/dokumentacje/Płatności/Płatności%20CashBill%232.3.pdf). TypeScript and Promises based.

## Installation

    npm i cashbill --save

## Features

- ✅ Retrieve existing payment
- ✅ Create new payment
- ✅ Validate payment response
